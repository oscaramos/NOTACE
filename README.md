# NOTACE
NOTACE: No Te acerques

Es un proyecto Arduino hecho para emitir una alarma(Buzzer pasivo) sonora cuando capte movimiento de un PIR(Sensor Infrarrojo). 

# Herramientas
* Arduino UNO
* Buzzer pasivo
* PIR
